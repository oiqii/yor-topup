Cara Melakukan Pemesanan

  1. Pilih game yang ingin kamu top up pada halaman Beranda
  2. Masukkan User ID & Zone ID secara benar
  3. Pilih nominal top up sesuai kebutuhan kamu
  4. Pilih metode pembayaran (Dana, OVO, Gopay, Bank)
  5. Tekan tombol Order via WhatsApp
  6. Kirim bukti pembayaran ke admin
  7. Diamond/UC akan masuk dalam ±1-10 menit
